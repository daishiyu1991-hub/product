# 供应链就绪度检查清单

## 一、供应商评估标准

### 1.1 核心评估维度

| 维度 | 评估要点 | 关键问题 |
|------|---------|---------|
| 产品质量 | 样品质量、一致性、缺陷率 | 样品是否符合预期？批量生产能否保持一致？ |
| 产能规模 | 最大月产能、扩产能力 | 能否满足增长需求？旺季能否加急生产？ |
| 最低起订量 (MOQ) | 首单MOQ、续单MOQ | MOQ是否匹配测试阶段需求？可否谈判降低？ |
| 交货周期 (Lead Time) | 生产周期、样品周期 | 标准交期多少天？加急交期和费用？ |
| 沟通能力 | 英语水平、响应速度、主动性 | 能否在24小时内回复？是否有专属业务员？ |
| 资质认证 | ISO、BSCI、相关行业认证 | 是否有目标市场所需的合规认证？ |
| 贸易经验 | 出口经验、亚马逊卖家服务经验 | 是否了解FBA标签和包装要求？ |
| 定制能力 | 模具开发、材料调整、设计实现 | 能否实现产品差异化设计？ |
| 价格竞争力 | 报价水平、议价空间 | 价格是否有竞争力？批量折扣阶梯？ |
| 合作稳定性 | 经营年限、客户评价、工厂规模 | 是否有倒闭/跑路风险？ |

### 1.2 供应商信息核实清单

- [ ] 通过1688/阿里巴巴平台验证工厂信息
- [ ] 确认是工厂直营还是贸易公司
- [ ] 查看工厂营业执照和经营范围
- [ ] 要求提供现有客户参考（如允许）
- [ ] 通过第三方（如SGS、TÜV）进行工厂审核（高价值订单）
- [ ] 索取产品合规测试报告（如适用）
- [ ] 确认出口资质和报关能力

## 二、供应商评分矩阵

### 2.1 评分标准 (1-5分)

| 分数 | 含义 | 描述 |
|------|------|------|
| 5 | 优秀 | 远超预期，行业领先水平 |
| 4 | 良好 | 满足需求且有一定优势 |
| 3 | 合格 | 基本满足要求 |
| 2 | 不足 | 存在明显短板，需改善 |
| 1 | 差 | 严重不达标，不建议合作 |

### 2.2 加权评分矩阵

| 评估维度 | 权重 | 供应商A | 供应商B | 供应商C |
|---------|------|--------|--------|--------|
| 产品质量 | 25% | _/5 | _/5 | _/5 |
| 价格竞争力 | 20% | _/5 | _/5 | _/5 |
| 交货周期 | 15% | _/5 | _/5 | _/5 |
| 沟通能力 | 15% | _/5 | _/5 | _/5 |
| 产能与MOQ | 10% | _/5 | _/5 | _/5 |
| 资质认证 | 8% | _/5 | _/5 | _/5 |
| 定制能力 | 7% | _/5 | _/5 | _/5 |
| **加权总分** | **100%** | **_/5** | **_/5** | **_/5** |

**决策阈值：**
- **≥ 4.0：** 推荐合作，优先选择
- **3.5-4.0：** 可以合作，关注短板
- **3.0-3.5：** 需改善后再考虑
- **< 3.0：** 不建议合作

### 2.3 评分注意事项

- 首单建议至少评估3家供应商
- 在评分前务必获取实物样品并进行对比
- 对关键维度（质量、交期）设定最低分门槛（如不得低于3分）
- 定期（每半年或每次采购后）更新评分

## 三、双供应商策略

### 3.1 何时建立备用供应商

| 触发条件 | 紧迫性 | 说明 |
|---------|-------|------|
| 月销稳定超过500件 | 高 | 单一供应商断供风险显著 |
| 年采购额超过$50,000 | 高 | 经济损失风险大 |
| 产品是核心盈利SKU | 高 | 影响整体业务健康度 |
| 供应商多次延迟交货 | 中高 | 可靠性下降信号 |
| 供应商产能接近上限 | 中高 | 无法支撑增长 |
| 进入旺季前 | 中 | 降低旺季断货风险 |
| 计划拓展新市场 | 中 | 多站点需要更大产能 |
| 供应商所在地区有政策/自然风险 | 中 | 地缘风险对冲 |

### 3.2 双供应商管理原则

```
主力供应商：承担70-80%订单量
备用供应商：承担20-30%订单量

关键要求：
1. 备用供应商需完成至少1次小批量实单验证
2. 两家供应商的产品规格/模具需兼容或统一
3. 备用供应商至少每季度下一次订单，保持关系活跃
4. 对两家供应商的品质标准、包装要求必须完全一致
5. 不建议向供应商透露对方信息
```

### 3.3 供应商切换检查

- [ ] 备用供应商样品已通过质检
- [ ] 包装和标签规格已共同确认
- [ ] 价格和MOQ已谈妥
- [ ] 至少完成一次小批量（50-100件）实单生产
- [ ] 实单产品已完成抽检或全检
- [ ] 物流路线和交期已确认
- [ ] 合同/订单条款已明确

## 四、质量控制体系

### 4.1 生产前检查 (Pre-Production Inspection)

**时机：** 原材料采购完成、正式量产开始前

**检查清单：**
- [ ] 原材料是否符合规格要求（材质、颜色、尺寸）
- [ ] 原材料供应商证书/检测报告是否齐全
- [ ] 生产样品（金样/封样）是否已确认并留存
- [ ] 包装材料（内包装、外箱、说明书、标签）是否到位
- [ ] FNSKU标签/条码是否正确打印
- [ ] 生产流程和工艺是否与样品一致
- [ ] 特殊要求（如定制颜色、logo位置）是否已传达到产线
- [ ] 交货日期和数量是否再次确认

### 4.2 生产中检查 (During Production Inspection / DUPRO)

**时机：** 生产完成约20-30%时

**检查要点：**
- [ ] 产线是否按照确认的工艺流程生产
- [ ] 已产出产品的外观、功能是否符合封样
- [ ] 关键尺寸是否在公差范围内
- [ ] 产品缺陷率是否在可接受范围内
- [ ] 包装方式是否正确（装箱数量、防护措施）
- [ ] 生产进度是否符合交期计划
- [ ] 是否存在系统性质量问题（需要停线纠正的）

**发现问题时的行动：**
```
轻微问题 → 记录并要求产线纠正，继续生产
严重问题 → 要求停线整改，确认纠正方案后再恢复
批量性缺陷 → 评估已产出产品，考虑返工或报废
```

### 4.3 出货前检查 (Pre-Shipment Inspection / PSI)

**时机：** 生产完成80-100%，装箱完毕后

#### AQL抽样标准 (Acceptable Quality Level)

**常用AQL水平：**

| 检查类别 | AQL水平 | 说明 |
|---------|---------|------|
| 严重缺陷 (Critical) | 0 | 安全隐患，零容忍 |
| 主要缺陷 (Major) | 2.5 | 影响功能或外观 |
| 次要缺陷 (Minor) | 4.0 | 轻微瑕疵 |

**抽样数量参考（一般检查水平II）：**

| 批量大小 | 抽样数量 | Major(AQL 2.5)允收/拒收 | Minor(AQL 4.0)允收/拒收 |
|---------|---------|----------------------|----------------------|
| 151-280 | 32 | 2/3 | 3/4 |
| 281-500 | 50 | 3/4 | 5/6 |
| 501-1200 | 80 | 5/6 | 7/8 |
| 1201-3200 | 125 | 7/8 | 10/11 |
| 3201-10000 | 200 | 10/11 | 14/15 |

**检查内容：**
- [ ] 外观检查（颜色、表面、印刷、组装）
- [ ] 尺寸测量（关键尺寸抽测）
- [ ] 功能测试（每一件抽样产品均需测试）
- [ ] 包装检查（内包装、外包装、标签、条码）
- [ ] 跌落测试（外箱跌落测试）
- [ ] 装箱数量核实
- [ ] 总箱数和总数量与订单核对
- [ ] 唛头和出货标签验证

### 4.4 第三方质检服务

| 服务商 | 特点 | 适用阶段 | 费用参考 |
|-------|------|---------|---------|
| SGS | 全球最大，公信力高 | 大批量/高价值 | $250-$400/人天 |
| Bureau Veritas | 欧洲认可度高 | 出口欧洲 | $250-$380/人天 |
| Intertek | 测试能力强 | 需要合规测试 | $250-$400/人天 |
| QIMA (AsiaInspection) | 线上下单方便 | 日常验货 | $189-$299/人天 |
| 国内独立验货师 | 灵活、性价比高 | 小批量/日常 | $100-$200/人天 |

**选择建议：**
- 首批订单：建议使用第三方PSI
- 日常订单（合作稳定后）：可由自己团队或独立验货师执行
- 高价值/高风险订单：使用知名第三方机构

## 五、库存管理体系

### 5.1 安全库存 (Safety Stock)

**公式：**
```
SS = Z × σ_d × √LT
```

**变量说明：**
| 变量 | 含义 | 说明 |
|------|------|------|
| Z | 服务水平系数 | 95%服务水平 → Z=1.65；99% → Z=2.33 |
| σ_d | 日销量标准差 | 反映销量波动程度 |
| LT | 补货提前期(天) | 从下单到入仓的总天数 |

**示例计算：**
```
日均销量: 10件
日销量标准差: 3件
补货提前期: 45天（生产30天 + 海运15天）
服务水平: 95% (Z=1.65)

SS = 1.65 × 3 × √45
   = 1.65 × 3 × 6.71
   = 33.2 ≈ 34件

即需要保持约34件安全库存以应对不确定性
```

**简化经验法则：**
```
安全库存 ≈ 日均销量 × 7-14天

新品期（数据不足）: 取14天
稳定期（数据充足）: 取7-10天
旺季前: 取14-21天
```

### 5.2 再订货点 (Reorder Point)

**公式：**
```
ROP = (日均销量 × 补货提前期) + 安全库存
```

**示例计算：**
```
日均销量: 10件
补货提前期: 45天
安全库存: 34件

ROP = (10 × 45) + 34 = 484件

当FBA可售库存降至484件时，应立即下达补货订单
```

**补货提前期组成：**
```
总提前期 = 下单确认(1-3天) + 生产周期(15-45天) + 质检(2-3天)
         + 内陆运输(2-5天) + 出口报关(1-3天) + 国际运输(5-40天)
         + 进口清关(1-5天) + FBA入仓(5-14天)

空运总提前期: 约25-40天
海运总提前期: 约50-80天
```

### 5.3 经济订购量 (EOQ) 简化版

**公式：**
```
EOQ = √(2 × D × S / H)
```

**变量说明：**
| 变量 | 含义 | 示例 |
|------|------|------|
| D | 年需求量 | 3,600件 (日均10 × 360天) |
| S | 单次订购成本 | $500（含下单、物流安排、验货等） |
| H | 单位年持有成本 | $3.00（仓储费 + 资金成本 + 损耗） |

**示例：**
```
EOQ = √(2 × 3600 × 500 / 3.00)
    = √(1,200,000)
    ≈ 1,095件

建议每次订购约1,100件，一年约订购3-4次
```

**实际应用调整：**
- EOQ结果需与MOQ做比较，取两者较大值
- 考虑现金流承受能力
- 考虑FBA仓储限制
- 旺季前可适当增加采购量

### 5.4 季节性库存规划

| 阶段 | 时间 | 行动 |
|------|------|------|
| 旺季分析 | 旺季前3-4个月 | 分析历史数据，预测旺季销量 |
| 备货下单 | 旺季前2.5-3个月 | 按预测量120-150%下单生产 |
| 首批入仓 | 旺季前1.5-2个月 | 首批库存到达FBA |
| 补货入仓 | 旺季前0.5-1个月 | 第二批库存到达FBA |
| 旺季运营 | 旺季期间 | 监控库存，必要时空运补货 |
| 季末清理 | 旺季后1个月 | 评估库存，促销清理滞销品 |

**Amazon重要时间节点：**
```
Prime Day: 通常7月中 → 5月前完成备货
返校季: 8-9月 → 6月前完成备货
黑五/网一: 11月下旬 → 9月前完成备货
圣诞: 12月 → 10月前完成备货
```

## 六、物流优化

### 6.1 空运 vs 海运 vs 快递 对比

| 对比维度 | 快递(Express) | 空运(Air Freight) | 海运(Sea Freight) |
|---------|-------------|-------------------|-------------------|
| 时效 | 3-7天 | 7-12天 | 25-45天 |
| 费用(kg) | $6-$10/kg | $3-$6/kg | $0.3-$1.0/kg |
| 适合重量 | < 100kg | 100-500kg | > 500kg / > 2CBM |
| 最低起运量 | 无限制 | 约45kg | 约1CBM或1吨 |
| 适用场景 | 样品、紧急补货、小批量 | 中等批量、较紧急 | 大批量、非紧急 |
| 优点 | 速度最快，门到门 | 较快，适合中等货量 | 成本最低 |
| 缺点 | 费用最高 | 费用中等，体积重限制 | 时间长，需要提前规划 |
| 报关 | 快递代理报关 | 需报关 | 需报关 |

### 6.2 空运转海运时机判断

**转换阈值参考：**
```
当满足以下条件时，应考虑从空运切换到海运：
1. 单次发货量 ≥ 500kg 或 ≥ 2CBM
2. 补货周期可预测且稳定
3. 库存深度足以覆盖海运在途时间
4. 海运成本比空运节省 ≥ 50%

经济性对比示例：
发货量: 1000kg
空运费用: $4/kg × 1000 = $4,000
海运费用: $0.60/kg × 1000 = $600
节省: $3,400 (85%)

月发货2次 → 年节省: $3,400 × 24 = $81,600
```

**过渡策略：**
```
阶段1 (新品期): 100%快递/空运 → 快速验证，灵活调整
阶段2 (增长期): 70%空运 + 30%海运 → 开始混合运输
阶段3 (稳定期): 80%海运 + 20%空运(补货) → 成本最优化
阶段4 (成熟期): 90%海运 + 10%空运(应急) → 完全切换
```

### 6.3 货运代理评估标准

| 评估维度 | 权重 | 评分标准 |
|---------|------|---------|
| 价格竞争力 | 25% | 对比3家以上报价 |
| 时效可靠性 | 25% | 准时率 ≥ 90% |
| 清关能力 | 20% | FBA清关经验、问题处理能力 |
| 沟通响应 | 15% | 在线时间、回复速度 |
| 附加服务 | 10% | 贴标、换标、分仓、仓储 |
| 赔偿保障 | 5% | 丢货/损坏赔偿条款 |

**选择建议：**
- 至少比较3家货运代理
- 优先选择有FBA发货经验的代理
- 确认是否提供亚马逊预约入仓服务
- 了解关税预付和垫付政策
- 首次合作用小批量测试

### 6.4 贸易术语解释

#### DDP (Delivered Duty Paid) - 完税后交货

```
卖方责任: 从工厂到买方指定地点的所有费用，包括关税和进口税
买方责任: 几乎无
适用场景: 买方希望简化操作，一口价采购
优点: 对买方最简单，价格透明
缺点: 卖方报价通常含溢价，总成本可能偏高
```

#### EXW (Ex Works) - 工厂交货

```
卖方责任: 在工厂交货，之后所有费用和风险由买方承担
买方责任: 运输、报关、关税、保险等所有费用
适用场景: 买方有成熟的物流链，希望完全控制物流
优点: 采购价最低，物流成本可控
缺点: 买方需处理所有物流和报关事务
```

#### FOB (Free on Board) - 离岸价

```
卖方责任: 负责到出口港装船，包括出口报关
买方责任: 海运费、进口报关、关税、目的港到FBA仓库的运输
适用场景: 最常见的中间方案，买方自安排海运
优点: 责任划分清晰，买方可选择具竞争力的船公司
缺点: 买方需协调进口环节
推荐: 大多数亚马逊卖家采购建议使用FOB条款
```

**选择建议：**
```
新手/小批量 → DDP（省心）
有经验/中等批量 → FOB（主流选择）
资深/大批量 → EXW + 自有物流链（成本最优）
```

## 七、FBA库存限制与仓储策略

### 7.1 FBA库存管理要点

| 管理指标 | 说明 | 健康范围 |
|---------|------|---------|
| IPI (库存绩效指数) | 综合评估库存健康度 | ≥ 400 (避免限制) |
| 冗余库存率 | 超过90天无销售的库存占比 | < 5% |
| 售罄率 | 有销量但库存不足的SKU占比 | 尽量接近0% |
| 滞留库存 | 需要处理的问题库存 | 及时处理 |

### 7.2 仓储优化策略

**降低仓储成本：**
- 使用亚马逊仓储费用计算器提前估算
- Q4前清除滞销库存，避免旺季高仓储费
- 利用"创建移除订单"功能处理超龄库存
- 考虑使用第三方海外仓作为中转站

**结合第三方海外仓的混合模式：**
```
策略: 大批量海运到海外仓 → 分批转运到FBA

优势:
1. 降低FBA仓储费（特别是旺季）
2. 避免FBA容量限制
3. 补货更灵活（海外仓转FBA仅需2-5天）
4. 降低断货风险

适用条件:
- 月销量稳定且可预测
- 愿意承担海外仓租赁费用
- 有能力管理双仓库存
```

## 八、风险缓解策略

### 8.1 供应中断风险

| 风险类型 | 概率 | 影响 | 缓解措施 |
|---------|------|------|---------|
| 工厂产能不足 | 中 | 高 | 建立备用供应商；提前下单 |
| 原材料涨价 | 中高 | 中 | 锁价合同；多次报价对比 |
| 工厂倒闭/转型 | 低 | 极高 | 双供应商策略；保留模具所有权 |
| 自然灾害/疫情 | 低 | 极高 | 跨地区供应商；增加安全库存 |
| 政策变化(关税等) | 中 | 高 | 关注政策动态；评估替代产地 |
| 春节/长假停工 | 确定 | 中 | 节前2个月多备1-2个月库存 |


**常见应急场景：**

**场景1：断货在即**
```
触发: FBA库存 < 7天销量 且补货在途 > 14天
行动:
1. 评估是否空运补货（计算成本）
2. 适当提价5-10%减缓销量
3. 降低广告出价和预算
4. 联系海外仓/三方仓是否有应急库存
预算: 空运加急费用不超过该SKU月利润的50%
```

**场景2：质检发现批量缺陷**
```
触发: PSI检查中Major缺陷率超过AQL标准
行动:
1. 拒收该批货物
2. 与工厂沟通返工方案和时间表
3. 评估返工后是否需要重新PSI
4. 如返工不可行，评估降价处理或报废
5. 启动备用供应商报价和打样
预算: 根据货值评估，原则上不接受有缺陷产品
```
