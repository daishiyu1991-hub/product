# 单位经济模型参考文档

## 一、完整成本结构分解

### 1.1 成本组成总览

| 成本项目 | 类型 | 典型占比(售价%) | 说明 |
|---------|------|-----------------|------|
| 商品成本 (COGS) | 固定/可变 | 25-40% | 采购价 + 包装 + 标签 + 质检 |
| 头程运费 (Shipping) | 可变 | 5-15% | 工厂到FBA仓库的运输费用 |
| FBA配送费 (FBA Fees) | 可变 | 8-15% | 亚马逊拣货、包装、配送费 |
| 佣金 (Referral Fee) | 可变 | 8-15% | 按品类收取的销售佣金 |
| 广告费 (Advertising) | 可变 | 5-20% | PPC广告及其他推广费用 |
| 退货成本 (Returns) | 可变 | 2-5% | 退货处理 + 损耗 + 退款 |
| 仓储费 (Storage) | 可变 | 1-5% | FBA月度仓储 + 超龄附加费 |
| 其他费用 | 可变 | 1-3% | 促销、coupon、vine等 |

### 1.2 COGS（商品成本）细分

```
COGS = 出厂价 + 内包装 + 外包装 + 产品标签 + FNSKU标签 + 质检费用(分摊)
```

**注意事项：**
- 出厂价应包含所有原材料和制造成本
- 包装成本常被低估，需包括：产品包装、说明书、感谢卡、防护材料
- 首批订单的模具费/开版费应单独核算，不计入单位COGS
- 汇率波动需预留3-5%缓冲

### 1.3 头程运费明细

```
单件头程运费 = 总运费 / 装箱数量

总运费 = 运输费 + 报关费 + 关税 + 保险 + 目的港费用
```

**关税计算：**
```
关税 = 申报价值 × 关税税率
到岸价值 (CIF) = FOB价 + 运费 + 保险费
```

## 二、核心公式体系

### 2.1 毛利率 (Gross Margin)

```
毛利率 = (售价 - COGS - FBA配送费 - 佣金 - 头程运费) / 售价 × 100%
```

**示例计算：**
```
售价: $29.99
COGS: $6.00 (20.0%)
FBA费: $5.50 (18.3%)
佣金: $4.50 (15.0%)
头程: $2.00 (6.7%)

毛利 = ($29.99 - $6.00 - $5.50 - $4.50 - $2.00) / $29.99
     = $11.99 / $29.99
     = 40.0%
```

### 2.2 净利率 (Net Margin)

```
净利率 = 毛利率 - TACoS - 退货成本率 - 仓储成本率 - 其他费用率
```

**示例计算（续）：**
```
毛利率: 40.0%
TACoS: -12.0%
退货:  -3.0%
仓储:  -2.0%
其他:  -1.5%

净利率 = 40.0% - 12.0% - 3.0% - 2.0% - 1.5%
       = 21.5%
```

### 2.3 贡献利润 (Contribution Margin)

```
贡献利润 = 售价 - 所有可变成本

可变成本 = COGS + FBA费 + 佣金 + 头程 + 广告费(单位) + 退货成本(单位) + 仓储费(单位)
```

**用途：** 评估每卖出一件产品对覆盖固定成本和利润的贡献。

### 2.4 盈亏平衡点 (Break-even Volume)

```
盈亏平衡销量 = 固定成本 / 单位贡献利润
```

**固定成本包括：**
- 产品开发费用（模具、打样、设计）
- 摄影和listing制作
- 品牌注册费用
- 初期库存投入的资金成本
- 软件/工具订阅费

**示例：**
```
固定成本总投入: $5,000（产品开发 + 上架准备）
单位贡献利润: $8.50
盈亏平衡销量 = $5,000 / $8.50 ≈ 589 件

若月销100件，需约6个月回本
若月销300件，需约2个月回本
```

### 2.5 客户获取成本 (CAC)

```
CAC = 广告总支出 / 通过广告获得的订单数

也可表达为：
CAC = ACoS × 平均售价
```

**其中：**
```
ACoS (广告销售成本比) = 广告支出 / 广告销售额
TACoS (总广告销售成本比) = 广告支出 / 总销售额（含自然订单）
```

**健康CAC评估：**
- CAC < 单位贡献利润的30% → 优秀
- CAC = 单位贡献利润的30-50% → 可接受
- CAC > 单位贡献利润的50% → 需要优化

### 2.6 客户终身价值 (LTV)

```
LTV = 平均订单价值 × 购买频率 × 客户生命周期

简化版（适用于复购品类）：
LTV = 单次利润 × 复购次数

复购适用品类判断：
- 消耗品（补充剂、清洁用品、美容产品）→ 高复购
- 耐用品（工具、家具、电子产品）→ 低复购
- 季节品（节日装饰、季节服装）→ 年度复购
```

**示例（消耗品）：**
```
单次净利润: $8.00
年均复购: 4次
客户生命周期: 2年
LTV = $8.00 × 4 × 2 = $64.00

可接受CAC ≤ LTV的25% = $16.00
```

## 三、FBA费用结构参考

### 3.1 标准尺寸 (Standard-Size)

| 尺寸分层 | 重量范围 | 配送费(约) |
|---------|---------|-----------|
| 小号标准 | ≤ 12 oz | $3.22-$3.40 |
| 大号标准 (1) | 12oz - 1lb | $3.86-$4.08 |
| 大号标准 (2) | 1-2 lb | $4.25-$5.40 |
| 大号标准 (3) | 2-3 lb | $5.40-$6.10 |
| 大号标准 (4) | 3-20 lb | $6.10-$8.90+ |

**尺寸限制（标准）：**
- 最长边 ≤ 18英寸
- 最短边 ≤ 8英寸
- 中间边 ≤ 14英寸
- 重量 ≤ 20磅

### 3.2 大件/超大件 (Oversize)

| 尺寸分层 | 配送费(约) |
|---------|-----------|
| 小号大件 | $9.73-$13.00+ |
| 中号大件 | $19.05-$25.00+ |
| 大号大件 | $89.98-$158.00+ |
| 特大件 | 单独报价 |

> **重要提示：** FBA费率每年调整（通常在1月和6月），务必查阅亚马逊最新费率卡。以上仅为参考范围。

### 3.3 费用优化策略

- **包装优化：** 减小包装尺寸可能降低费用层级，节省$0.50-$2.00/件
- **重量优化：** 使用轻质包装材料，避免跨越重量阈值
- **多件装（Multipack）：** 提高客单价的同时摊薄单件FBA费
- **尺寸分层边界：** 在设计阶段就考虑FBA尺寸分层，避免刚好超标

## 五、仓储费用明细

### 5.1 月度仓储费

| 时期 | 标准尺寸 (每立方英尺) | 大件 (每立方英尺) |
|------|---------------------|------------------|
| 1-9月 (非旺季) | $0.78 | $0.56 |
| 10-12月 (旺季) | $2.40 | $1.40 |

### 5.2 超龄库存附加费 (Aged Inventory Surcharge)

| 库龄 | 附加费 (每立方英尺/月) |
|------|---------------------|
| 271-300天 | $1.50 |
| 301-330天 | $3.80 |
| 331-365天 | $5.45 |
| 365天以上 | $6.90+ |

### 5.3 仓储费优化建议

- **目标库龄：** 保持90%以上库存在180天以内
- **补货节奏：** 小批量、高频次补货优于大批量囤货
- **旺季策略：** 10月前清理滞销库存，为旺季畅销品腾出空间
- **移除策略：** 超龄前提前做促销清仓或移除，避免高额附加费
- **库存绩效指数(IPI)：** 保持400+以上，避免仓储限制

## 六、健康基准范围

### 6.1 净利率基准

| 等级 | 净利率范围 | 评估 | 行动建议 |
|------|-----------|------|---------|
| 优秀 | ≥ 25% | 高利润产品 | 加大投入，快速扩张 |
| 良好 | 15-25% | 健康利润 | 稳定运营，持续优化 |
| 可接受 | 10-15% | 尚可但需关注 | 寻找成本优化空间 |
| 警告 | 5-10% | 利润微薄 | 必须优化或考虑淘汰 |
| 危险 | < 5% | 几乎无利润 | 立即评估是否继续 |

### 6.2 TACoS基准

| 等级 | TACoS范围 | 评估 | 含义 |
|------|----------|------|------|
| 优秀 | ≤ 8% | 品牌力强 | 自然流量占比高，广告依赖低 |
| 良好 | 8-12% | 健康状态 | 广告和有机流量平衡良好 |
| 一般 | 12-18% | 需要关注 | 广告依赖偏高 |
| 偏高 | 18-25% | 需要优化 | 过度依赖广告，利润受压 |
| 过高 | > 25% | 危险信号 | 严重依赖广告，需紧急调整 |

### 6.3 其他关键指标基准

| 指标 | 优秀 | 良好 | 警告 |
|------|------|------|------|
| ACoS | < 15% | 15-30% | > 30% |
| 退货率 | < 3% | 3-8% | > 8% |
| 转化率 (CVR) | > 15% | 10-15% | < 10% |
| 库存周转天数 | < 30天 | 30-60天 | > 90天 |
| Review评分 | ≥ 4.3 | 4.0-4.3 | < 4.0 |

## 七、敏感性分析模板

### 7.1 单变量敏感性分析

**基准场景：**
```
售价: $29.99 | COGS: $6.00 | 月销: 300件 | 净利率: 21.5%
```

| -10% | $26.99 | 15.6% | -$850 | 接近可接受下限 |
| -5% | $28.49 | 18.5% | -$430 | 仍在健康范围 |
| 基准 | $29.99 | 21.5% | $0 | - |
| +5% | $31.49 | 24.2% | +$390 | 需关注竞争力 |
| +10% | $32.99 | 26.8% | +$760 | 可能影响转化率 |

| COGS变动 | 新COGS | 预估净利率 | 月净利润变化 | 备注 |
|---------|--------|-----------|------------|------|

| +15% | $6.90 | 18.5% | -$270 | 需要谈判或换供应商 |
| +10% | $6.60 | 19.5% | -$180 | 可接受范围 |
| 基准 | $6.00 | 21.5% | $0 | - |
| -10% | $5.40 | 23.5% | +$180 | 议价或规模效应 |

#### 广告效率变动影响

### 7.2 多变量组合分析

**最坏情况组合：** 价格-10% + COGS+15% + TACoS+50%
```

```

**最佳情况组合：** 价格+5% + COGS-10% + TACoS-25%
```
净利率 ≈ 30.2% → 高利润，加速投入
```

## 八、多场景建模

### 8.1 三场景模型

#### 悲观场景 (Pessimistic)

| 假设 | 数值 |
|------|------|
| 月销量 | 150件 (-50%) |
| 售价 | $27.99 (-7%) |
| COGS | $6.60 (+10%) |
| TACoS | 18% (+50%) |
| 退货率 | 6% (高) |
| **预估净利率** | **6.2%** |
| **月净利润** | **$260** |

**触发条件：** 强劲竞品入场、价格战、供应链涨价、季节性低谷
**应对策略：** 降低广告预算，专注利润；考虑供应链优化；评估是否退出

#### 基准场景 (Baseline)

| 假设 | 数值 |
|------|------|
| 月销量 | 300件 |
| 售价 | $29.99 |
| COGS | $6.00 |
| TACoS | 12% |
| 退货率 | 3% |
| **预估净利率** | **21.5%** |
| **月净利润** | **$1,935** |

**基于：** 当前数据和合理预期

#### 乐观场景 (Optimistic)

| 假设 | 数值 |
|------|------|
| 月销量 | 500件 (+67%) |
| 售价 | $31.99 (+7%) |
| COGS | $5.40 (-10%) |
| TACoS | 8% (-33%) |
| 退货率 | 2% (低) |
| **预估净利率** | **30.8%** |
| **月净利润** | **$4,928** |

**触发条件：** 品牌力提升、竞品退出、旺季、review积累到位
**前置条件：** 库存充足、listing优化到位、广告精细化运营

### 8.2 场景概率加权预期

```
加权月利润 = (悲观利润 × 20%) + (基准利润 × 60%) + (乐观利润 × 20%)
           = ($260 × 0.2) + ($1,935 × 0.6) + ($4,928 × 0.2)
           = $52 + $1,161 + $986
           = $2,199/月

年度预期利润 ≈ $2,199 × 12 = $26,388
```

### 8.3 投资回报评估

```
总投入 = 产品开发 + 首批库存 + 上架准备 + 前3个月运营预算
预估总投入 = $2,000 + $3,600 + $1,500 + $3,000 = $10,100

基准场景回本周期 = $10,100 / $1,935 ≈ 5.2个月
悲观场景回本周期 = $10,100 / $260 ≈ 38.8个月 (不可接受)
乐观场景回本周期 = $10,100 / $4,928 ≈ 2.0个月

加权回本周期 = $10,100 / $2,199 ≈ 4.6个月
```

## 九、决策框架

### 9.1 Go/No-Go 经济性判断

| 条件 | Go | 谨慎 | No-Go |
|------|-----|------|-------|
| 基准净利率 | ≥ 15% | 10-15% | < 10% |
| 悲观场景净利率 | > 5% | 0-5% | 亏损 |
| 回本周期(基准) | < 6个月 | 6-12个月 | > 12个月 |
| 投资总额风险 | < 月收入20% | 20-50% | > 50% |
| 价格下降10%后利润 | 仍盈利 | 微利 | 亏损 |

### 9.2 规模化经济性检查

**从验证到规模化时需重新评估：**
- [ ] 大批量采购COGS是否有5-15%下降空间
- [ ] 海运替代空运后头程成本下降幅度
- [ ] TACoS趋势是否在持续下降
- [ ] 退货率是否稳定或下降
- [ ] 是否有涨价空间（review和品牌力支撑）
- [ ] 竞争环境是否允许维持当前价格
- [ ] 季节性波动对年均利润的影响
