# Amazon 广告策略模板

## 概述

本文档提供 Amazon PPC 广告的标准化策略模板，涵盖关键词分层、广告架构、竞价策略、预算分配、优化流程等。所有模板基于精益迭代理念设计，强调数据驱动和持续优化。

---

## 一、关键词分层体系

### 1.1 四层关键词架构

将所有关键词按照战略功能分为四个层级，每层采用不同的投放策略：

#### 🎯 收割词（Harvest Keywords）

- **定义**：已验证的高转化关键词，ACOS 低于目标值
- **来源**：广告报告中 ACOS < 目标值且订单 ≥ 5 的搜索词
- **匹配方式**：精确匹配（Exact Match）
- **竞价策略**：激进竞价，争取 Top of Search 展示位
- **目标**：最大化已验证词的销售贡献，保持或提升自然排名
- **关键指标**：ACOS < 目标值、自然排名稳定在前 3 页

#### 🌱 培养词（Nurture Keywords）

- **定义**：有潜力但尚未充分验证的关键词
- **来源**：广告报告中有点击和少量转化的词、关键词工具中的高相关性词
- **匹配方式**：词组匹配（Phrase Match）
- **竞价策略**：中等竞价，给予足够曝光来积累数据
- **目标**：积累转化数据，达标后升级为收割词
- **关键指标**：点击量是否足够判断（建议 ≥ 20 次点击后评估）

#### 🛡️ 防御词（Defense Keywords）

- **定义**：自有品牌词和核心 ASIN 相关词
- **来源**：品牌名称、品牌变体、自有产品型号
- **匹配方式**：精确匹配
- **竞价策略**：中低竞价（品牌词 CPC 通常较低），确保首位展示
- **目标**：防止竞品抢占品牌流量，保护品牌搜索份额
- **关键指标**：品牌搜索份额（Brand Share of Voice）

#### ⛔ 否定词（Negative Keywords）

- **定义**：确认无效的、不相关的、持续亏损的关键词
- **来源**：高花费零转化词、不相关搜索词、低转化率词
- **匹配类型**：
  - **Negative Exact**：仅屏蔽该精确搜索词
  - **Negative Phrase**：屏蔽包含该短语的所有搜索词
- **管理规则**：
  - 花费 > 2 倍产品售价且零转化 → 加入否定精确
  - 明显不相关的搜索词 → 加入否定词组
  - 竞品品牌名（非 ASIN 定向时）→ 加入否定词组

### 1.2 关键词生命周期流转

```
【发现阶段】                【验证阶段】              【成熟阶段】
自动广告/广泛匹配   →    培养词（Phrase）   →    收割词（Exact）
     ↓                        ↓                       ↓
  否定词 ←──── 数据不达标 ────┘         持续监控与维护
```

**升级标准**（培养词 → 收割词）：
- 点击次数 ≥ 20
- 转化率 ≥ 类目平均
- ACOS ≤ 目标 ACOS

**降级/淘汰标准**（→ 否定词）：
- 点击次数 ≥ 20 且零转化
- ACOS 持续 > 目标值 2 倍以上
- 与产品明显不相关

## 二、广告活动结构模板

### 2.1 自动广告活动（Auto Campaign）

自动广告是关键词发现的核心引擎，按匹配类型拆分为独立广告组以实现精细化控制。

#### 广告活动结构

```
📂 [SP] Auto - [产品名/ASIN]
  ├── 📁 广告组 1: Close Match（紧密匹配）
  │     ├── 竞价：建议竞价的 100-120%
  │     ├── 目标：发现与产品高度相关的新关键词
  │     └── 说明：匹配与产品属性紧密相关的搜索词
  │
  ├── 📁 广告组 2: Loose Match（宽泛匹配）
  │     ├── 竞价：建议竞价的 80-100%
  │     ├── 目标：拓展相关但非直接匹配的关键词
  │     └── 说明：匹配与产品间接相关的搜索词
  │
  ├── 📁 广告组 3: Substitutes（替代品）
  │     ├── 竞价：建议竞价的 80-100%
  │     ├── 目标：出现在竞品产品详情页
  │     └── 说明：定向到与你产品相似的竞品 ASIN
  │
  └── 📁 广告组 4: Complements（互补品）
        ├── 竞价：建议竞价的 60-80%
        ├── 目标：出现在互补产品详情页
        └── 说明：定向到与你产品互补的产品 ASIN
```

#### 自动广告管理要点

- 每周检查搜索词报告，将高转化词迁移至手动广告
- 将不相关词加入否定关键词列表
- 保持自动广告持续运行，作为关键词发现的常备引擎
- 定期降低已迁移至手动广告的重复关键词竞价

---

### 2.2 手动广告活动（Manual Campaign）

按匹配类型拆分为独立广告活动，实现差异化竞价和预算控制。

#### 精确匹配活动（Exact Match）

```
📂 [SP] Manual Exact - [产品名/ASIN]
  ├── 📁 广告组: Harvest（收割词）
  │     ├── 关键词：已验证的高转化精确关键词
  │     ├── 竞价：积极竞价，使用 Top of Search 调整 (+20-50%)
  │     ├── 预算：占手动广告总预算的 40-50%
  │     └── 目标：最大化销售额，维持自然排名
  │
  └── 📁 广告组: Brand Defense（品牌防御）
        ├── 关键词：品牌名 + 品牌变体词
        ├── 竞价：中低竞价
        ├── 预算：占手动广告总预算的 5-10%
        └── 目标：保护品牌流量不被竞品截取
```

#### 词组匹配活动（Phrase Match）

```
📂 [SP] Manual Phrase - [产品名/ASIN]
  └── 📁 广告组: Nurture（培养词）
        ├── 关键词：潜力关键词的词组匹配
        ├── 竞价：中等竞价，建议竞价的 90-110%
        ├── 预算：占手动广告总预算的 20-30%
        ├── 否定词：已在精确匹配活动中投放的词（避免自我竞争）
        └── 目标：积累数据，发现新的可收割关键词
```

#### 广泛匹配活动（Broad Match）

```
📂 [SP] Manual Broad - [产品名/ASIN]
  └── 📁 广告组: Discovery（拓展发现）
        ├── 关键词：核心关键词的广泛匹配 + 修饰广泛匹配
        ├── 竞价：保守竞价，建议竞价的 70-90%
        ├── 预算：占手动广告总预算的 10-20%
        ├── 否定词：已在精确/词组匹配活动中投放的词
        └── 目标：最大化关键词覆盖面，发现长尾机会
```

### 2.3 品牌广告活动（Sponsored Brands）

#### SB 标题搜索广告（Headline Search）

```
📂 [SB] Headline - [产品线名称]
  ├── 📁 广告组 1: Category Keywords
  │     ├── 关键词：类目核心词（精确+词组）
  │     ├── 创意：品牌 Logo + 标题文案 + 3 个产品
  │     ├── 着陆页：品牌旗舰店对应类目页
  │     └── 竞价：较高竞价（SB 展示位竞争激烈）
  │
  └── 📁 广告组 2: Brand Keywords
        ├── 关键词：品牌词（精确匹配）
        ├── 创意：品牌故事文案 + 热卖产品展示
        ├── 着陆页：品牌旗舰店首页
        └── 竞价：中等竞价
```

#### SB 视频广告（Sponsored Brands Video）

```
📂 [SB Video] - [产品名/ASIN]
  └── 📁 广告组: Core Keywords
        ├── 关键词：核心高转化关键词（精确+词组）
        ├── 创意：15-30 秒产品展示视频
        ├── 着陆页：产品详情页
        ├── 竞价：中高竞价（视频广告 CTR 通常较高）
        └── 视频要求：
            - 时长 15-30 秒
            - 前 3 秒抓住注意力
            - 展示产品核心卖点和使用场景
            - 添加字幕（静音播放优化）
            - 分辨率 1920x1080 或 1280x720
```

### 2.4 展示广告活动（Sponsored Display）

#### 受众定向（Audiences）

```
📂 [SD] Audiences - [产品名/ASIN]
  ├── 📁 广告组 1: Remarketing Views
  │     ├── 定向：过去 30 天浏览过自己产品但未购买的用户
  │     ├── 竞价：中高竞价（高意向用户）
  │     └── 目标：再营销促转化
  │
  ├── 📁 广告组 2: Remarketing Purchases
  │     ├── 定向：过去购买过自己品牌产品的用户
  │     ├── 竞价：中等竞价
  │     └── 目标：交叉销售、复购
  │
  └── 📁 广告组 3: In-Market Audiences
        ├── 定向：正在浏览相关类目的用户
        ├── 竞价：保守竞价（用户意向较低）
        └── 目标：品牌曝光，漏斗上层引流
```

#### 商品定向（Product Targeting）

```
📂 [SD] Product Targeting - [产品名/ASIN]
  ├── 📁 广告组 1: Competitor ASINs
  │     ├── 定向：核心竞品 ASIN（选择评分低于自己的竞品）
  │     ├── 竞价：中高竞价
  │     └── 目标：从竞品页面截取流量
  │
  └── 📁 广告组 2: Category Targeting
        ├── 定向：相关类目，可按价格/评分/品牌筛选
        ├── 竞价：保守竞价
        └── 目标：扩大产品曝光面
```

## 三、竞价策略

### 3.1 按关键词类型的竞价矩阵

| 关键词类型 | 基础竞价 | Top of Search 调整 | Product Pages 调整 | 竞价策略 |
|-----------|---------|-------------------|-------------------|---------|
| 收割词（Exact） | 建议竞价 120-150% | +20% 至 +50% | +0% 至 +10% | 动态竞价 - 提升和降低 |
| 培养词（Phrase） | 建议竞价 90-110% | +0% 至 +20% | +0% | 动态竞价 - 仅降低 |
| 拓展词（Broad） | 建议竞价 70-90% | +0% | +0% | 动态竞价 - 仅降低 |
| 品牌词（Exact） | 建议竞价 80-100% | +10% 至 +30% | +0% | 动态竞价 - 仅降低 |
| 自动广告 Close | 建议竞价 100-120% | +0% 至 +20% | +0% | 动态竞价 - 仅降低 |
| 自动广告 Loose | 建议竞价 80-100% | +0% | +0% | 固定竞价 |

### 3.2 竞价调整规则

#### 提高竞价的条件

- ACOS < 目标 ACOS 的 70% 且展示量不足
- 转化率 > 类目平均，但展示位靠后
- 旺季/大促前需要抢占高位展示
- 新品推广期需要快速积累订单

#### 降低竞价的条件

- ACOS > 目标 ACOS 的 130%，持续 7 天
- 点击率（CTR）< 0.3%，说明展示位不匹配
- 已获得稳定的自然排名 Top 3

#### 竞价调整幅度

- 单次调整幅度不超过 ±20%
- 每次调整后至少观察 3-5 天再做下一次调整
- 大幅调整（>30%）分 2-3 次阶梯式完成

## 四、预算分配模板

### 4.1 按广告类型分配

| 广告类型 | 预算占比 | 适用阶段 | 主要目标 |
|---------|---------|---------|---------|
| SP 自动广告 | 15-20% | 全阶段 | 关键词发现 |
| SP 手动精确 | 35-40% | 全阶段 | 核心转化 |
| SP 手动词组 | 15-20% | 全阶段 | 关键词培养 |
| SP 手动广泛 | 5-10% | 全阶段 | 长尾拓展 |
| SB 标题/视频 | 10-15% | 品牌建设期 | 品牌曝光 |
| SD 展示广告 | 5-10% | 成熟期 | 再营销/防御 |

### 4.2 按产品生命周期调整

#### 新品期（第 1-4 周）

```
总预算：较高（积极投资获取数据）
SP 自动广告：30%   ← 重点：快速发现关键词
SP 手动精确：30%   ← 投放已验证的核心词
SP 手动词组：20%   ← 拓展更多关键词
SP 手动广泛：10%   ← 长尾挖掘
SB/SD：10%         ← 少量品牌曝光
目标 ACOS：可接受较高 ACOS（30-50%），以换取排名
```

#### 成长期（第 5-12 周）

```
总预算：中高（持续投资但追求效率）
SP 自动广告：20%   ← 持续但减少比例
SP 手动精确：40%   ← 重点：收割已验证词
SP 手动词组：15%   ← 持续培养
SP 手动广泛：5%    ← 减少比例
SB：10%            ← 增加品牌广告
SD：10%            ← 启动再营销
目标 ACOS：接近盈亏平衡 ACOS（20-35%）
```

#### 成熟期（第 13 周+）

```
总预算：优化（追求利润最大化）
SP 自动广告：15%   ← 维持最低发现功能
SP 手动精确：40%   ← 核心转化引擎
SP 手动词组：10%   ← 维持
SP 手动广泛：5%    ← 维持
SB：15%            ← 品牌建设
SD：15%            ← 再营销+竞品防御
目标 ACOS：< 盈亏平衡 ACOS（15-25%）
```

## 五、否定关键词管理流程

### 5.1 否定关键词添加流程

```
第 1 步：下载搜索词报告（过去 7-14 天）
    ↓
第 2 步：筛选条件
    ├── 条件 A：花费 > 产品售价 × 2 且 订单 = 0 → 候选否定词
    ├── 条件 B：点击 ≥ 20 且 转化率 = 0 → 候选否定词
    └── 条件 C：搜索词与产品明显不相关 → 立即否定
    ↓
第 3 步：确认否定类型
    ├── 单个无效词 → Negative Exact（否定精确）
    └── 含有无效词根的一系列搜索词 → Negative Phrase（否定词组）
    ↓
第 4 步：添加否定词到对应广告活动
    ├── 活动级别否定：应用于整个广告活动
    └── 广告组级别否定：仅应用于特定广告组
    ↓
第 5 步：记录否定词及原因（用于复查）
```

### 5.2 否定关键词维护规则

- **每周一次**：检查自动广告和广泛匹配活动的搜索词报告
- **每月一次**：全面审查所有否定关键词列表，移除可能误否的词
- **每季度一次**：评估否定词对流量和销售的累积影响

### 5.3 常见否定词类别

| 类别 | 示例 | 否定类型 |
|------|------|---------|
| 竞品品牌名 | "anker", "baseus" | Negative Phrase |
| 不相关材质 | "wooden"（你卖金属款） | Negative Phrase |
| 不相关用途 | "for car"（你卖家用款） | Negative Phrase |
| 超低价意图 | "cheap", "free" | Negative Phrase |
| DIY/批发意图 | "wholesale", "bulk" | Negative Phrase |
| 不同尺寸 | "large"（你只卖 small） | Negative Exact |

## 六、ACOS 优化工作流

### 6.1 ACOS 诊断决策树

```
当前 ACOS > 目标 ACOS？
├── 是 → 分析原因
│   ├── CTR 低 (< 0.3%) → 广告展示位不佳或创意问题
│   │   ├── 检查竞价是否过低
│   │   ├── 检查主图吸引力
│   │   └── 检查标题关键词相关性
│   │
│   ├── CVR 低 (< 类目均值) → Listing 转化力不足
│   │   ├── 检查定价竞争力
│   │   ├── 检查评论数量和评分
│   │   ├── 检查 Listing 内容质量
│   │   └── 检查是否有缺货/变体问题
│   │
│   └── CPC 高 → 竞价过高或竞争激烈
│       ├── 降低竞价 10-15%
│       ├── 转移预算到长尾词
│       └── 增加否定关键词减少浪费
│
└── 否 → ACOS < 目标值 50%？
    ├── 是 → 可能投放过于保守
    │   ├── 适当提高竞价获取更多流量
    │   └── 拓展新关键词
    └── 否 → 维持当前策略，持续监控
```

### 6.2 ACOS 分级管理

| ACOS 范围 | 状态 | 行动 |
|-----------|------|------|
| < 目标值 50% | 投放过保守 | 提高竞价、扩展关键词 |
| 目标值 50-100% | 健康盈利 | 维持并适度扩展 |
| 目标值 100-150% | 轻度超标 | 优化竞价和否定词 |
| 目标值 150-200% | 严重超标 | 削减竞价、暂停低效词 |
| > 目标值 200% | 严重亏损 | 暂停重新评估、大幅调整 |

### 6.3 TACOS（Total ACOS）追踪

```
TACOS = 广告总花费 / 总销售额（广告+自然）× 100%

理想趋势：
- 随着自然排名提升，TACOS 应逐步下降
- 新品期 TACOS 15-25% 可接受
- 成熟期 TACOS 应控制在 8-12%
- TACOS 持续上升说明广告依赖度过高，需要优化自然排名
```

## 七、分时段策略（Day-Parting）

### 7.1 分时段分析方法

1. 下载过去 30 天的广告报告（按小时维度）
2. 统计每个小时的花费、销售额、ACOS
3. 识别高转化时段（ACOS < 目标值）和低效时段（ACOS > 目标值 2 倍）

### 7.2 分时段执行策略

#### 基础策略（适合大多数卖家）

| 时段（目标市场当地时间） | 竞价调整 | 原因 |
|------------------------|---------|------|
| 06:00 - 09:00 | +0%（标准） | 早间浏览高峰开始 |
| 09:00 - 12:00 | +10-20% | 上午购物高峰 |
| 12:00 - 14:00 | +5-10% | 午间小高峰 |
| 14:00 - 18:00 | +0%（标准） | 下午平稳期 |
| 18:00 - 22:00 | +10-20% | 晚间购物高峰 |
| 22:00 - 01:00 | -10-20% | 深夜浏览减少 |
| 01:00 - 06:00 | -20-30% | 凌晨低效时段 |

#### 进阶策略（使用第三方工具自动化）

- 使用 Day-Parting 工具（如 Perpetua、Pacvue）实现自动化分时竞价
- 根据自己的实际数据定制时段配置
- 周末与工作日可设置不同策略
- 大促期间关闭分时段限制，全天最高竞价

### 7.3 注意事项

- Amazon 不原生支持精确到小时的预算控制
- 分时段策略主要通过第三方工具的竞价调整实现
- 手动操作的替代方案：在低效时段手动降低每日预算
- 确保不会因为分时段限制导致高转化时段预算提前耗尽

## 九、每周优化例程

### 9.1 每周优化日历

#### 周一：数据回顾与规划

```
□ 下载过去 7 天的广告报告
□ 汇总核心指标：花费、销售额、ACOS、TACOS
□ 对比上周数据，标注异常波动
□ 制定本周优化重点（1-2 个核心动作）
```

#### 周二：搜索词分析与关键词管理

```
□ 分析搜索词报告
□ 高转化搜索词 → 迁移至手动精确匹配
□ 无效搜索词 → 添加否定关键词
□ 更新关键词分层表（收割/培养/否定）
□ 检查关键词索引状态
```

#### 周三：竞价与预算优化

```
□ 调整高/低效关键词的竞价
□ 检查各活动的预算消耗情况
□ 重新分配预算（从低效活动到高效活动）
□ 检查 Top of Search 展示份额变化
□ 记录竞价调整内容和原因
```

#### 周四：竞品与市场分析

```
□ 检查核心竞品的广告展示位变化
□ 监控自己的 Share of Voice
□ 分析竞品的新增广告创意
□ 检查类目整体 CPC 趋势
□ 评估是否需要调整竞争策略
```

#### 周五：创意与 Listing 联动

```
□ 检查 SB/SD 广告创意的表现
□ 评估是否需要更新广告创意
□ 检查 Listing 变更对广告效果的影响
□ 准备下周的 A/B 测试计划
□ 记录本周优化行动和效果总结
```

### 9.2 每周核心关注指标

| 指标 | 健康范围 | 关注信号 | 行动 |
|------|---------|---------|------|
| ACOS | < 目标 ACOS | 连续 7 天超标 | 调竞价/加否定词 |
| TACOS | < 12% | 连续上升趋势 | 检查自然排名 |
| CTR | > 0.4% | < 0.3% | 优化主图/标题 |
| CVR | > 类目均值 | 持续下降 | 优化 Listing |
| CPC | 稳定 | 突然上升 > 20% | 检查竞争环境 |
| 花费/销售额 | 预算内 | 提前耗尽预算 | 调整分配 |
| 展示量 | 稳定/增长 | 突然下降 > 30% | 检查竞价/预算 |

### 9.3 月度回顾（每月最后一周额外执行）

```
□ 本月广告总投资回报（ROAS）计算
□ 各广告活动的 Lifetime 表现评估
□ 关键词组合的覆盖率分析
□ 广告对自然排名的拉动效果评估
□ 下月广告策略和预算规划
□ 清理长期未活跃的广告活动
□ 更新广告架构（是否需要新增/合并活动）
```

## 附录：关键公式速查

```
ACOS = 广告花费 / 广告销售额 × 100%
ROAS = 广告销售额 / 广告花费
TACOS = 广告花费 / 总销售额 × 100%
盈亏平衡 ACOS = (售价 - 成本) / 售价 × 100%
目标 ACOS = 盈亏平衡 ACOS × (1 - 目标利润率/毛利率)
CPC = 广告花费 / 点击次数
CTR = 点击次数 / 展示次数 × 100%
CVR = 订单数 / 点击次数 × 100%

每日预算估算 = 目标日均订单数 / CVR × CPC
```
