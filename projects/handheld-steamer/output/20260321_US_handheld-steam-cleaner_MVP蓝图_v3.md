## Step 3: 痛点根因拆解

### 产品工作原理简图

```
 安全盖(按压式) ← 故障点A: 密封圈 NBR 耐温不足
     │
 ┌───┴────┐
 │ 水 箱  │ 350ml (有效仅250ml)
 │  水面  │ ← 故障点E: 无水位可视
 └───┬────┘
     │
 ┌───┴────┐
 │ 加热腔 │ 1050W 铸铝, 115°C, 2-3bar
 │        │ ← 故障点B: 温控死区25°C→间歇出汽
 └───┬────┘
     │ (无气液分离) ← 故障点C: 液态水直出=喷水
     │
  蒸汽阀 (弹簧复位, 需持续按压)
     │
  Push-fit接口 ← 故障点D: 保持力<15N, 蒸汽推力12.5N→弹飞
     │
  喷嘴/刷头 → 蒸汽 115°C

  电源: 120V → 1050W → 9.4ft线
  外壳: PP (热变形~100°C < 蒸汽115°C) ← 故障点F: 开裂
  手柄: 硬塑料, 无隔热 ← 故障点G: 过热烫手
```

### 差评频次排序（CURMEDI, ~100条差评样本）

| 排名 | 物理现象 | 频次 | 根因类型 |
|------|---------|------|---------|
| 1 | 蒸汽弱/清洁力差 | ~28 次 | ❌ **品类限制**（1050W = 120V/15A 物理极限） |
| 2 | 喷水/吐水 | ~22 次 | ✅ 设计缺陷（无气液分离结构） |
| 3 | 配件脱落 | ~20 次 | ✅ 设计缺陷（push-fit 保持力不足） |
| 4 | 停止工作/加热失效 | ~18 次 | ✅ 设计缺陷（一次性 TCO 熔断） |
| 5 | 水箱太小/续航差 | ~16 次 | ⚠️ 混合（喷水浪费 + 品类体积限制） |
| 6 | 烫伤/安全 | ~10 次 | ✅ 设计缺陷（配件弹飞+手柄过热+无浪涌限流） |
| 7 | 壳体开裂 | ~10 次 | ✅ 设计缺陷（PP 耐温不足） |
| 8 | 蒸汽间歇 | ~9 次 | ✅ 设计缺陷（温控死区过大） |
| 9 | 手柄过热 | ~7 次 | ✅ 设计缺陷（无隔热层） |
| 10 | 跳闸 | ~2 次 | ✅ 设计缺陷（无浪涌限流） |

### 关键根因 5-Whys（只列最重要的 3 个）

**根因 C — 喷水（频次 #2, ~22次）：**
```
喷水 → 液态水从喷嘴出 → 水未完全汽化 → 加热腔出口无气液分离
→ 根因: 缺少多孔挡板/螺旋通道, 蒸汽裹挟液滴直出
→ 方案: 蒸汽出口加 3 层不锈钢多孔挡板(孔径1-2mm) [+¥3-5]
```

**根因 D — 配件弹飞（频次 #3, ~20次）：**
```
弹飞烫伤 → 蒸汽推出力12.5N > push-fit保持力8N(磨损后)
→ 根因: 连接方式选型错误, push-fit 不适合加压系统
→ 方案: Snap-Lock 卡扣, POM 卡爪, 保持力≥50N [+¥3-5]
```

**根因 3A — 蒸汽弱（频次 #1, ~28次）：**
```
清洁力差 → 蒸汽产出仅30-50g/min → 1050W物理极限 → 120V/15A美国家用
→ 类型: ❌ 品类限制, 无法根本解决
→ 应对: Listing 定位"daily cleaning, chemical-free", 不承诺"deep clean"
         + 优化喷嘴口径提高冲击力 + 铜刷物理辅助
```
