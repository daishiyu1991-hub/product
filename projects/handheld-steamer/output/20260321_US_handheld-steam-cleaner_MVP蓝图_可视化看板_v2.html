产品工作原理 & 故障点定位
先搞清楚产品怎么工作，才能定位故障发生在哪个环节
痛点根因拆解
"漏水"不是一个问题，而是 4 种完全不同的物理故障。每种故障需要不同的工程解决方案。
根因 1A
冷水喷射 Water Spitting
设计缺陷 — 可改善 ✅
喷水 → 液态水未汽化 → 加热腔无气液分离 → 蒸汽裹挟液滴直出
MVP 方案
蒸汽出口增加 3 层不锈钢多孔挡板（孔径 1-2mm）+ 水箱 MAX 250ml 水位线刻蚀
+¥3-5.5/件
根因 1B
密封圈老化泄漏 Seal Degradation
设计缺陷 — 可改善 ✅
盖子漏热水 → 密封圈失弹 → NBR 耐温仅 100°C < 蒸汽 115°C → 热降解+压缩永久变形+挤出
MVP 方案
密封圈换 EPDM（耐温 150°C+）+ 双沟槽双圈 + 旋锁盖(bayonet) + 备用圈 ×2
+¥4-8/件
根因 1C
壳体开裂 Housing Crack
设计缺陷 — 可改善 ✅
裂开 → PP 热疲劳 → PP 热变形温度 ~100°C < 蒸汽 115°C → 材质等级不足+缺加强筋
MVP 方案
升级 PP+30%GF（热变形 150°C+），加热腔壁厚≥3mm，应力集中区加强筋
+¥2-4/件
根因 2
配件弹飞 Attachment Pop-off
设计缺陷 — 可改善 ✅
配件弹飞烫伤 → 推出力 12.5N > 保持力 8N → Push-fit 磨损后过盈减小 → 连接方式选型错误
MVP 方案
Snap-Lock 卡扣（POM 卡爪，保持力≥50N），侧面释放按钮防误脱
+¥3-5/件
根因 3A
蒸汽绝对弱 Power Limitation
品类限制 — 无法根本解决 ❌
清洁力差 → 蒸汽产出仅 30-50g/min → 1050W 物理极限 → 120V/15A = max 1800W
应对策略（非解决）
Listing 定位"日常轻量清洁"而非"deep cleaning" | 优化喷嘴口径提高冲击力 | 铜刷物理辅助
无法通过成本解决
根因 3B
蒸汽间歇中断 Intermittent Steam
设计缺陷 — 可改善 ✅
蒸汽停停走走 → 温控断电阶段无蒸汽 → 死区 25°C 太大 + NTC 位置偏 → 温控器规格+安装位置
MVP 方案
温控死区缩至 10°C（115°C 断-108°C 复），NTC 移至靠近水面处
+¥1.5/件
根因 4A
启动跳闸 Circuit Breaker Trip
设计缺陷 — 可改善 ✅
插电跳闸 → 冷启动浪涌 17-26A → > 15A 断路器 → 无浪涌抑制
MVP 方案
电源入口串联 NTC 浪涌限流器（冷态高阻限流 → 热态低阻正常工作）
+¥1-2/件
根因 4B+6
手柄过热 & 握持酸痛
设计缺陷 — 可改善 ✅
烫手+酸痛 → 115°C 热传导至手柄 + 截面设计差 → 无隔热+人体工学缺失
MVP 方案
硅胶包覆 2-3mm + 壳体与手柄间 3-5mm 空气隔热层 + 椭圆截面
+¥3-5/件
根因 4C
加热元件早期失效
设计缺陷 — 可改善 ✅
用3次就坏 → 干烧触发 TCO 热熔断器 → 一次性断路永久失效 → 无可复位保护
MVP 方案
NTC 温控 + 可复位双金属片温控器双保险（替代一次性 TCO）
+¥2-3/件
根因 5
水箱续航差
混合 — 部分可改善 ⚠️
3分钟没水 → 大量水以液态喷出(非汽化消耗) + 加热效率仅~50% → 解决1A后自然缓解
MVP 方案
解决 1A 冷水喷射 + 透明水位窗减少焦虑（手持产品大水箱=品类限制）
+¥2-3/件
