# 飞轮诊断方法论

> 来源：梁宁《增长思维30讲》模式模块 — 增强回路与调节回路
> 核心：增长不是线性的，是飞轮式的。飞轮转起来 = 指数增长；飞轮卡住 = 增长停滞。

## 原理

### 增强回路（正向飞轮）
每一个环节的输出成为下一个环节的输入，且每转一圈都比上一圈更强。

### 调节回路（刹车系统）
防止飞轮失控的反馈机制。没有调节回路 = 增长失控 = 库存爆仓/现金流断裂/品质崩溃。

### 亚马逊产品飞轮模型

```
好产品 → 好 Review → 高转化率 → 高排名 → 更多流量 → 更多销量 → 更多 Review → ...
   ↑                                                              │
   └──────────────────────────────────────────────────────────────┘

调节回路（刹车）:
├── 库存管理：断货 = 飞轮停转
├── 利润底线：ACOS失控 = 烧钱换量
├── 品控标准：品质下降 = 差评螺旋
└── 广告预算：无上限投放 = 现金流危机
```

## 执行步骤

### Step 1: 数据采集

**已上架产品**（有ASIN）：
| 数据 | 工具 | 用途 |
|------|------|------|
| 产品详情 | `product_detail` | 基础信息 |
| 产品趋势 | `product_trend`（SalesVolume） | 销量飞轮是否在转 |
| 产品趋势 | `product_trend`（Rank） | 排名飞轮是否在转 |
| 正面评论 | `product_reviews`（Positive） | 好Review是否在产出 |
| 负面评论 | `product_reviews`（Negative） | 差评是否在增加 |
| 关键词排名 | `competitor_product_keywords` | 自然排名是否在提升 |
| 流量词 | `product_traffic_terms` | 流量来源是否在扩大 |

**规划中产品**（无ASIN）：
- 基于品类数据和竞品数据，设计预期飞轮
- 标注每个环节的假设和验证方法

### Step 2: 画出飞轮（5个关键节点）

```
节点1: 产品力
  ↓ 驱动
节点2: Review质量
  ↓ 驱动
节点3: 转化率
  ↓ 驱动
节点4: 搜索排名
  ↓ 驱动
节点5: 流量规模
  ↓ 回到
节点1: 更多销量 → 更多数据 → 产品迭代 → 产品力↑
```

### Step 3: 逐节点诊断

对每个节点评估：**正常运转 / 减速 / 断裂**

| 节点 | 正常信号 | 减速信号 | 断裂信号 |
|------|---------|---------|---------|
| **产品力** | 星级≥4.3，差评率<15% | 星级4.0-4.3，差评率15-25% | 星级<4.0，差评率>25% |
| **Review质量** | 近30天好评稳定产出 | 好评减少或差评增加 | 差评>好评，或出现差评螺旋 |
| **转化率** | 转化率≥品类均值 | 转化率略低于均值 | 转化率持续下降 |
| **搜索排名** | 核心词Top 3页稳定 | 排名波动/下滑 | 核心词跌出前3页 |
| **流量规模** | 曝光词数量稳定/增长 | 曝光词减少 | 仅靠广告维持，无自然流量 |

### Step 4: 找到断裂点

**诊断逻辑**：
- 如果产品力OK但Review差 → 断裂点在"产品力→Review"环节（可能是客服/售后问题）
- 如果Review好但转化差 → 断裂点在"Review→转化"环节（可能是Listing/主图/价格问题）
- 如果转化好但排名差 → 断裂点在"转化→排名"环节（可能是流量基数不够/广告不足）
- 如果排名好但流量不增 → 断裂点在"排名→流量"环节（可能是关键词覆盖不够/品类天花板）

### Step 5: 检查调节回路（刹车系统）

| 调节回路 | 健康标准 | 失控信号 | 后果 |
|---------|---------|---------|------|
| **库存管理** | 库存周转30-60天 | 断货或库存>90天 | 断货=飞轮停，积压=现金流冻结 |
| **利润底线** | ACOS<品类均值，TACOS<15% | ACOS持续上升，TACOS>20% | 烧钱换量，不可持续 |
| **品控标准** | 差评率<15%且稳定 | 差评率趋势性上升 | 差评螺旋→排名下滑→流量减少 |
| **广告依赖** | 自然订单占比>50% | 广告订单占比>70% | 一停广告就没销量=假飞轮 |
| **价格战底线** | 毛利率>30% | 被迫跟价降到毛利<20% | 价格螺旋→利润消失 |

### Step 6: 输出诊断报告

## 注意事项

1. **假飞轮识别**：如果广告关停后销量骤降>50%，说明飞轮是广告驱动的"假飞轮"
2. **差评螺旋是最致命的断裂**：差评→转化↓→排名↓→只剩价格敏感用户→更多差评
3. **调节回路不是坏事**：刹车是为了不翻车。每个飞轮都需要设置刹车阈值
4. **飞轮需要时间**：新品前3个月飞轮可能还没转起来，这是正常的启动期
