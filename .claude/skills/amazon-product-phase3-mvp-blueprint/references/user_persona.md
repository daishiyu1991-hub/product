# 用户画像构建方法论

## 为什么用户画像要在根因拆解之前？

用户画像决定了后续每一步的判断基准：

| 后续步骤 | 用户画像如何影响该步骤 |
|---------|---------------------|
| 根因拆解 | 同一个"蒸汽弱"痛点，专业用户觉得不可忍受(需加大功率)，轻量用户觉得够用(只需管理期望)。画像决定哪些痛点是真正的 deal-breaker |
| 创新亮点 | LED 灯对科技型用户有吸引力，对价格敏感用户是浪费成本。画像决定哪些创新值得投入 |
| 外观配色 | 薄荷绿吸引年轻女性家庭用户，深灰吸引男性/专业用户。画像决定配色方向 |
| 影响力排序 | 冲动购买型用户主图权重极高，研究型用户评价/参数权重更高 |
| Listing 文案 | 语气、卖点顺序、场景选择必须用用户的语言说用户关心的事 |
| 工业设计 | 传给下游 Phase 4/5 的上下文需要包含目标用户的审美偏好 |

## 数据来源

| 数据源 | 提取什么 | 怎么用 |
|--------|---------|--------|
| **搜索词语义** | 用户怎么搜这个产品？ | "steam cleaner for home" → 家用场景 |
| **竞品差评用户自述** | 使用场景和身份线索 | "senior citizen""for my kids" → 年龄/场景 |
| **竞品好评用户自述** | 真正在意什么 | "perfect for spot cleaning" → 轻量需求 |
| **品类平均价位** | $20-50 vs $100-200 | 价位段暗示消费能力和期望值 |
| **品类搜索趋势** | 旺季/增长趋势 | 旺季月份暗示使用场景 |

## 画像输出模板

```
## 目标用户画像

### 主力用户 (Primary Persona) — 占比 ~60-70%
- 人物标签：[如：25-45岁女性家庭主妇/职业女性]
- 核心诉求：[如：日常轻量清洁，不想用化学清洁剂]
- 购买决策模式：[如：TikTok 看到 → Amazon 搜索 → 主图+价格+评分 → 3分钟决策]
- 审美偏好：[如：清新、家居感、马卡龙色系]
- 价格敏感度：[如：$30-50 舒适区，>$60 犹豫]
- 痛点优先级：[如：安全 > 效果 > 便利]
- 使用场景：[如：厨房灶台/浴室瓷砖缝]

### 次要用户 (Secondary Persona) — 占比 ~20-30%
- [同上结构]

### 边缘用户 (Edge Case) — 占比 ~10%
- [不作为 MVP 核心目标]

### 画像对产品定义的决策影响
| 决策点 | 画像指导 |
|--------|---------|
| 配色选择 | 主力用户审美偏好 |
| 蒸汽弱是否要解决 | 主力用户核心诉求 |
| Listing 第一卖点 | 主力用户痛点优先级 |
| 定价策略 | 主力用户价格敏感度 |
```

## 没有数据时怎么办？

1. **从品类定价推断消费层级**：$20-50 入门 / $50-100 中端 / $100+ 专业
2. **WebSearch 搜索**："[product] buyer demographics"
3. **从竞品 Listing 风格推断**：竞品对谁说话？场景图里是什么人？
4. **标注为推测**：推断的画像必须标注 ⚠️
